# Visual Matrix UX Writing

Status: consolidacao de tom, labels e microcopy.

## Tom Geral

- Direto, operacional e calmo.
- Reduz carga cognitiva.
- Declara incerteza sem dramatizar.
- Ajuda decisao, nao julga produtividade.
- Evita motivacao generica, gamificacao e linguagem de performance.

## Proibicoes De Linguagem

- Nao usar produtividade como tese central.
- Nao usar score, performance, streak dominante ou ranking.
- Nao usar "vou fazer por voce" para IA.
- Nao chamar Inbox de backlog.
- Nao chamar Memoria de arquivo morto, wiki ou knowledge base.
- Nao reintroduzir Orbita, Ritual do Dia, Inteligencia como rota principal ou Explorar como territorio.

## Confirmacao IA

Toda microcopy de acao proposta por IA deve explicitar:

- o que sera alterado;
- qual entidade sera afetada;
- se ha informacao faltante;
- se a confianca e baixa;
- que a acao so persiste apos confirmacao humana.

## Exclusao De Conta

Texto de exclusao deve ser governado por Central: escopo, retencao, exportacao previa, falha segura e reversibilidade quando existir. Nao prometer delecao total sem backend validado.
